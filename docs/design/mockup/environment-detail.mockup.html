A HUD environment wrapping Browserbase for remote browser sessions. Runs a headless Chromium instance accessible via Playwright. Supports answer, navigation, form-fill, and game scenarios out of the box.

Scenarios (6)

Each scenario represents a specialized skill or task you can perform with this environment. Select one to configure and run it.

2048-near-win 11 runs

Navigate the 2048 game from a near-win state to a winning board. Tests multi-step planning with a known-solvable starting position.

todo-complete

Complete a percentage of seeded todos. Tests task execution with partial-completion grading against a target completion ratio.

todo-create

Create a new todo item with a specific title. Tests single-step form interaction with exact-match grading.
