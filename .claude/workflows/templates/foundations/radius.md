# Border Radius Tokens

```css
--radius-badge: 4px;   /* tx status badges, entity labels */
--radius-none:  0px;   /* flush table cell borders */
/* --radius-chip: 6px; — reuses --radius-sm, no new token needed */
```

## Philosophy

Angular = data, rounded = navigation. Tables and badges use 4px. Cards and dialogs use base `--radius-lg` (12px).
